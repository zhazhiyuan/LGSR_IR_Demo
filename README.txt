
This demo is using LGSR algorithm to Image Restoration.

The curret demo supports three image restoration tasks, including image denoising, image inpainting and image CS.


When you want to conduct the image denoising task, you can directly run the function "LGSR_Denoising_Demo.m".

When you want to conduct the image inpainting task, you can directly run the function "LGSR_Inpainting_Demo.m".

When you want to conduct the image denoising task, you can directly run the function "LGSR_CS_Demo.m".
